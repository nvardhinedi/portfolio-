Reflection 7/14/17

Today, we learned how to use whisker wires to allow the robot to navigate through a maze with obstacles. In order to use whisker wires, it is necessary to use a 3-pin header and two resistors per whisker wire: one that connects to the power and one that connects to a pin on the digital side. Even though we had some hardware issues with our robot, we were able to write the basic code needed for the robot to turn when the whisker wires detect obstacles.
